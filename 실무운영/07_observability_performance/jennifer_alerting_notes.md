# jennifer_alerting_notes

## 1. 개요
Jennifer APM에서 이메일 알림과 라이선스 교체를 수행할 때 필요한 운영 메모를 정리한 문서입니다.

## 2. 작업 배경
APM은 수집 자체보다 알림 체계가 실제 운영 대응 속도에 더 큰 영향을 줍니다.
이메일 알림 설정이 제대로 되어 있어야 장애 이벤트를 놓치지 않을 수 있습니다.

## 3. 핵심 정리
- SMTP 설정
- 발신자/수신자 지정
- Event Rule 기반 알림 연결
- 라이선스 키 교체 절차

## 4. 운영 체크포인트
- SMTP 포트와 인증 정책 확인
- 다수 수신자 구분 규칙 확인
- 알림 테스트 메일을 실제로 발송해볼 것
